Step-by-Step Installation

1

Preparation

  • ✓ Confirm installation location (minimum 1.5m from floor)
  • ✓ Check all components are present in packaging
  • ✓ Prepare Phillips screwdriver and level
  • ✓ Ensure power outlet is accessible
2

Handle Installation

  • ✓ Locate side handles in accessory kit
  • ✓ Align handles with pre-drilled holes on unit sides
  • ✓ Secure with provided screws (hand-tighten first)
  • ✓ Use screwdriver to fully tighten all screws

⚠️ CRITICAL: Handles must be installed BEFORE lifting unit from box

3

Mounting & Leveling

  • ✓ Using handles, lift unit from packaging (2-person recommended)
  • ✓ Position unit against wall at desired location
  • ✓ Use level to ensure unit is perfectly vertical
  • ✓ Mark wall mounting holes with pencil
  • ✓ Drill holes and install wall anchors
  • ✓ Secure unit to wall with provided mounting hardware
4

Power & Testing

  • ✓ Connect power cable to 120V outlet
  • ✓ Turn on main power switch (located on side)
  • ✓ Check LED status indicators (green = normal operation)
  • ✓ Test all fan speed settings (Low/Medium/High)
  • ✓ Connect to WiFi and configure via Ridgesun App (1000ie only)

Frequently Asked Questions

What if the unit doesn't power on?

Check: 1) Power outlet is working 2) Power cable is securely connected 3) Main switch is turned on. If issues persist, contact support.

Can I install it myself or do I need professional help?

The PureRing is designed for DIY installation. However, due to its weight (85 lbs), we recommend having a second person assist with lifting and mounting.

How do I know when to replace filters?

Standard replacement interval is 6 months for HEPA filter, 3 months for pre-filter. The 1000ie model provides filter life monitoring via the app.
